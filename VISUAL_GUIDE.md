## 預期視覺效果

### ✅ 你應該看到

1. **遊戲開始時**
   - 咖啡色麵包板背景
   - 20 個熄滅的 LED（4 × 5）
   - 4 個紅色按鈕在底部
   - 白色判定線

2. **音符出現時**
   - 對應軌道的 LED 開始依序亮起
   - 從上往下像流水一樣
   - 發光效果很明顯

3. **按下按鈕時**
   - 按鈕有 3D 按壓動畫
   - 判定文字彈出
   - 分數更新

### ❌ 如果畫面還是黑的

可能的原因：

1. **JavaScript 錯誤**
   - 按 F12 打開開發者工具
   - 查看 Console 是否有紅色錯誤訊息

2. **Canvas 未載入**
   - 重新整理頁面（Ctrl+R 或 Cmd+R）
   - 確認伺服器正在運行

3. **瀏覽器快取**
   - 硬性重新整理（Ctrl+Shift+R）
   - 或清除瀏覽器快取

## 自訂視覺

### 想改變 LED 顏色？

編輯 [public/js/game.js](public/js/game.js)，找到 `drawLED()` 函數：

```javascript
// 亮起時的顏色
ledGradient.addColorStop(0, '#ffffff');   // 中心白色
ledGradient.addColorStop(0.3, '#00ffff'); // 改這裡！
ledGradient.addColorStop(0.7, '#00d4ff'); // 改這裡！
ledGradient.addColorStop(1, '#0088cc');   // 改這裡！
```

### 想改變按鈕顏色？

編輯 `drawButton()` 函數：

```javascript
// 按鈕顏色
gradient.addColorStop(0, '#ff4444'); // 改這裡！
gradient.addColorStop(1, '#aa0000'); // 改這裡！
```

## 最佳觀賞設定

1. **全螢幕模式**（F11）
2. **關閉其他視窗**（減少干擾）
3. **調暗房間燈光**（LED 效果更明顯）
4. **使用大螢幕**（至少 1280×720）

## 故障排除

### LED 沒有亮起

**症狀**：音符下落但 LED 一直是灰色
**解決**：
1. 檢查 Console 是否有錯誤
2. 確認 `updateLEDStates()` 函數正常執行
3. 嘗試重新啟動伺服器

### 按鈕沒有按壓效果

**症狀**：按鍵沒有視覺回饋
**解決**：
1. 確認正在使用 D/F/J/K 鍵
2. 確認遊戲已經開始（不在選歌畫面）
3. 檢查 `lanePressTime` 是否正常更新

### 畫面很卡

**症狀**：FPS 低於 30
**解決**：
1. 關閉其他應用程式
2. 使用更好的瀏覽器（Chrome/Edge 推薦）
3. 降低 Canvas 解析度
